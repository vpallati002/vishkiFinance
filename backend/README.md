## Description
This project is built by using [Nest](https://github.com/nestjs/nest) framework.

## Install necessary tools globally

```bash
$ npm install -g yarn
$ npm i -g @nestjs/cli
```

## Package Installation

```bash
$ cd backend
$ yarn run app:install
```

For convenience, `nest` and `typeorm` commnads use path from node_modules; all commands in package.json are updated to use local installation

## App require following env variables to be set

```bash
TYPEORM_HOST
TYPEORM_USERNAME
TYPEORM_PASSWORD
TYPEORM_DATABASE
JWT_SECRET
JWT_EXPIRY_IN_SEC (Default: 600)
JWT_REFRESH_EXPIRY_IN_SEC (Default: 3600)
```

all the above process.env variables are set in `.env` file in the backend folder root

## Install PostgresQL 
```bash
Install locally or use RDS instance - URL: https://www.postgresql.org/download/windows/
Install pgAdmin GUI tool for PostgreSQL DB
make sure you create a database with the same name that you specified in above env variable: TYPEORM_DATABASE before you run the migration. 
```

## Running Migrations

```bash
$ yarn run migration:run
```
`The above command will bring the database to the latest migration`

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
